Testing the Server: In your terminal, run make to compile the project, then ./server to start it. Visit http://localhost:3490/d20 and http://localhost:3490/index.html in your browser.
Testing the Cache: Navigate to the src directory and run make tests to verify your LRU cache works correctly.
Best use in linux, when using windows use wsl or rewrite the code to include the required windows file to run on msys2(UCRT64)
